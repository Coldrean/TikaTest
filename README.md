#测试 Tika

##目前测试来看，Tika只能提取word中的文字，无法提取图片

##Word
###可提取的元信息
创建人
作者
文档字数
创建时间
标题
最后一次修改时间
最后一次打印时间
文档类型
页数

##输出形式
可以输出为xhtml文件，可以保留word中的表格结构。
可以提取页眉页脚
